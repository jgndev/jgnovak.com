<a href="https://loanshark.jgnovak.net" target="blank" rel="noreferrer">Loan Shark</a>
is a Loan Calculator built from two decoupled parts. A Back End REST API endpoint
written in <a href="https://go.dev" target="_blank" rel="noreferrer">Go</a> deployed to
<a href="https://learn.microsoft.com/en-us/azure/azure-functions/functions-get-started?pivots=programming-language-javascript"
   target="_blank" rel="noreferrer">Azure Functions</a>,
and a <a href="https://react.dev" target="_blank" rel="noreferrer">React</a> App written in
<a href="https://www.typescriptlang.org/" target="_blank" rel="noreferrer">TypeScript</a> deployed to
<a href="https://learn.microsoft.com/en-us/azure/static-web-apps/overview" target="_blank" rel="noreferrer">Azure Static
    Web Apps</a>.
